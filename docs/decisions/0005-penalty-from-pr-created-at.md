# ADR-0005: Штраф за дедлайн по дате открытия PR

**Status:** Accepted  
**Date:** 2026-03-20

## Context

Нужно определить момент "сдачи" для расчёта штрафа. Варианты: дата последнего коммита, дата открытия PR, дата первого push в ветку.

## Decision

Использовать `pull_request.created_at` из GitHub API.

## Rationale

- **Immutable**: GitHub API не позволяет менять `created_at` PR
- **Невозможно подменить**: в отличие от `git commit --date` (студент может подменить)
- **Однозначный**: один PR = одна дата, не зависит от количества коммитов
- **Доступен в webhook**: приходит в payload, не нужен доп. API вызов

## Consequences

- Студент не может подменить дату сдачи
- Если студент открыл PR рано и дорабатывал — штраф считается по первому открытию (справедливо: PR = заявка на проверку)
- Формула: 0 дней = 1.0, ≤1 день = 0.9, ≤3 дня = 0.7, ≤7 дней = 0.5, >7 = 0.2
